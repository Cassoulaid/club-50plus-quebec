## Technologies

**Tailwind CSS**

Pourquoi Tailwind ?
- Aucune installation requise
- Pas de configuration complexe
- Fonctionne rapidement
- Design moderne et cohérent
- Responsive par défaut


## Lancement rapide

**Prérequis**

Node.js installé (v20 minimum)

**Lancement**

- Cloner le dépôt GitHub : ``git clone https://github.com/Cassoulaid/club-50plus-quebec ``
- Installer les dépendances : ``npm init -y``
- Lancer Tailwind : ``npm install -D tailwindcss @tailwindcss/cli``
- Ouvrir index.html dans le navigateur ou via un serveur local (Live Server)
